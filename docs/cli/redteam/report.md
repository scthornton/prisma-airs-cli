# redteam report

## redteam report

View scan report

```text
airs redteam report [options] <jobId>
```

### Arguments

- `jobId` (required) —

### Options

| Flag | Required | Default | Description |
|------|:--------:|---------|-------------|
| `--attacks` | No | `false` | Include attack list |
| `--severity <level>` | No | — | Filter attacks by severity |
| `--limit <n>` | No | `20` | Max attacks to show |
